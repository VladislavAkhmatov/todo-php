# Todo List на PHP

Это простой проект Todo List, разработанный на PHP с использованием объектно-ориентированного программирования (ООП), автозагрузки классов, библиотеки `symfony/var-dumper` для отладки, подключения к базе данных, выполнения запросов к базе данных, роутинга с помощью библиотеки `AltoRouter` и минималистичного дизайна.

## Особенности проекта

- **ООП**: Проект построен с использованием объектно-ориентированного подхода, что делает код более структурированным и поддерживаемым.
- **Autoloader**: Для автоматической загрузки классов используется встроенный автозагрузчик PHP (Composer).
- **symfony/var-dumper**: Используется для удобного вывода отладочной информации.
- **Подключение к базе данных**: Проект подключается к базе данных MySQL для хранения задач.
- **Запросы к базе данных**: Выполняются SQL-запросы для создания, чтения, обновления и удаления задач (CRUD).
- **Роутинг**: Для маршрутизации используется библиотека `AltoRouter`, которая позволяет легко управлять маршрутами.
- **Дизайн**: Простой и минималистичный дизайн для удобства использования.

## Установка и настройка

1. **Клонируйте репозиторий**:
   git clone https://github.com/VladislavAkhmatov/todo-php.git
   cd todo-php

2. **Установите зависимости**:
   Убедитесь, что у вас установлен Composer. Затем выполните:

   **composer install**

3. **Настройте базу данных**:

       Создайте базу данных MySQL.

       Импортируйте SQL-файл **todo.sql** в папке db:

4. **Настройте подключение к базе данных**:
   Откройте файл src/config.php и укажите параметры подключения к вашей базе данных:
   php
   Copy

   define('DB_HOST', 'localhost');
   define('DB_NAME', 'todo');
   define('DB_USER', 'ваш_пользователь');
   define('DB_PASS', 'ваш_пароль');

5. **Запустите встроенный PHP-сервер**:

   **php -S localhost:8000 -t public**

   Теперь проект будет доступен по адресу http://localhost:8000.
